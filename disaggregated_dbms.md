## A reading list for disaggregated DBMS

### Foundational cloud native architecture

[Aurora: Design Considerations for High Throughput Cloud Native Relational Databases (SIGMOD 2017)](https://dl.acm.org/doi/epdf/10.1145/3035918.3056101)

[Aurora: On Avoiding Distributed Consensus for I/Os Commits and Membership Changes (SIGMOD 2018)](https://dl.acm.org/doi/epdf/10.1145/3183713.3196937)

[Socrates: The New SQL Server in the Cloud (SIGMOD 2019)](https://dl.acm.org/doi/epdf/10.1145/3299869.3314047)

[PolarDB Serverless: A Cloud Native Database for Disaggregated Data Centers (SIGMOD 2021)](https://dl.acm.org/doi/epdf/10.1145/3448016.3457560)

### Disaggregated storage

[Disaggregating Persistent Memory and
Controlling Them Remotely: An Exploration
of Passive DisaggregatedKey-Value Stores (USENIX 2020)](https://www.usenix.org/system/files/atc20-tsai.pdf)

[The Snowflake Elastic Data Warehouse](https://dl.acm.org/doi/epdf/10.1145/2882903.2903741)

### Distributed storage foundations

[Dynamo: Amazon's Highly Available Key Value Store (SOSP 2007)](https://dl.acm.org/doi/epdf/10.1145/1323293.1294281)

[Spanner: Google's Globally Distributed Database (OSDI 2012)](https://dl.acm.org/doi/pdf/10.1145/2491245)

[Bigtable: A Distributed Storage System for Structured Data (ACM TOCS 2008)](https://dl.acm.org/doi/epdf/10.1145/1365815.1365816)

[In search of an understandable consensus algorithm (USENIX 2014)](https://raft.github.io/raft.pdf)

[TiDB: A Raft-based HTAP Database (VLDB 2020)](https://www.vldb.org/pvldb/vol13/p3072-huang.pdf)

[F1 Lightning: HTAP as a Service (VLDB 2020)](https://pages.cs.wisc.edu/~yxy/cs764-f20/papers/f1-lightning.pdf)

[HyPer (ICDE 2011)](https://cs.brown.edu/courses/cs227/archives/2012/papers/olap/hyper.pdf)
