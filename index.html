CHIP 8 Interpreter

C++

A simple emulator program that can run CHIP 8 games on Mac. CHIP 8 runs on a virtual machine, meaning the stack, memory and registers are all virtual (no actual hardware). CHIP 8 uses only 8 bit instructions, hence the "8".
